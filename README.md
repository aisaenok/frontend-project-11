RSS Aggregator — учебное веб-приложение для подписки на RSS-потоки и чтения постов.

## Requirements

- Node.js >= 18
- npm >= 9

## Installation

```bash
git clone https://github.com/aisaenok/frontend-project-11.git
cd frontend-project-11
make install
```
